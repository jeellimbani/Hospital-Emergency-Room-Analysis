# 🏥 Hospital-Emergency-Room-Analysis
Excel dashboard project analyzing hospital emergency room performance, patient flow, and operational KPIs.

## 📌 Project Overview

This project analyzes Hospital Emergency Room data using Microsoft Excel. The dashboard provides insights into patient admissions, waiting times, satisfaction scores, department referrals, gender distribution, and age group analysis through interactive visualizations and slicers.

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- Slicers
- Conditional Formatting

## 📊 Dashboard Highlights

- Total Patients
- Average Waiting Time
- Patient Satisfaction Score
- Admission Status
- Patient Attend Status
- Gender-wise Analysis
- Department Referral Analysis
- Age Group Distribution
